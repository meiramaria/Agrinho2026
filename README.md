# Agrinho2026
O **“Fazenda Sustentável — Projeto Agrinho 2026”** é um site educativo interativo desenvolvido para o concurso Agrinho 2026, com o tema *“Do Campo à Mesa: Sustentabilidade em Cada Colheita”*. Ele tem como objetivo ensinar, de forma acessível, conceitos de agricultura sustentável ao público escolar.

O projeto apresenta uma jornada de aprendizado que inclui os fundamentos da sustentabilidade no campo (pilares ambiental, econômico e social), técnicas agrícolas sustentáveis como irrigação por gotejamento, compostagem, energia solar e controle biológico, além de dados sobre seus impactos. Também traz uma galeria visual, uma seção sobre tecnologias futuras no agronegócio (como drones, IoT e inteligência artificial) e um quiz interativo para fixação do conteúdo.

Todo o conteúdo é baseado em fontes institucionais como FAO, EMBRAPA, IPCC, MAPA e INPE. Tecnicamente, o site é construído com HTML, CSS e JavaScript puro, com design responsivo, animações e elementos interativos para melhorar a experiência do usuário.
O **Fazenda Sustentável — Projeto Agrinho 2026** funciona como um site educativo interativo em formato de “jornada de aprendizado”.

Ao acessar o site, o usuário navega por diferentes seções organizadas de forma progressiva. Primeiro, ele aprende os conceitos básicos de agricultura sustentável e seus três pilares: ambiental, econômico e social. Em seguida, conhece técnicas utilizadas no campo, como irrigação por gotejamento, compostagem, energia solar e controle biológico.

Depois disso, o site apresenta exemplos e dados sobre os impactos positivos dessas práticas, reforçando a importância da sustentabilidade na produção de alimentos. Em outra parte, o usuário explora o futuro da agricultura, com tecnologias como drones, sensores (IoT) e inteligência artificial.

Para tornar o aprendizado mais dinâmico, o projeto inclui elementos visuais, animações e uma galeria de imagens. No final, há um quiz interativo que testa o conhecimento adquirido durante a navegação.

Tudo isso é integrado em uma página responsiva feita com HTML, CSS e JavaScript, permitindo uma experiência fluida tanto no computador quanto no celular.
