# Agrinho2026
O **“Fazenda Sustentável — Projeto Agrinho 2026”** é um site educativo interativo desenvolvido para o concurso Agrinho 2026, com o tema *“Do Campo à Mesa: Sustentabilidade em Cada Colheita”*. Ele tem como objetivo ensinar, de forma acessível, conceitos de agricultura sustentável ao público escolar.

O projeto apresenta uma jornada de aprendizado que inclui os fundamentos da sustentabilidade no campo (pilares ambiental, econômico e social), técnicas agrícolas sustentáveis como irrigação por gotejamento, compostagem, energia solar e controle biológico, além de dados sobre seus impactos. Também traz uma galeria visual, uma seção sobre tecnologias futuras no agronegócio (como drones, IoT e inteligência artificial) e um quiz interativo para fixação do conteúdo.

Todo o conteúdo é baseado em fontes institucionais como FAO, EMBRAPA, IPCC, MAPA e INPE. Tecnicamente, o site é construído com HTML, CSS e JavaScript puro, com design responsivo, animações e elementos interativos para melhorar a experiência do usuário.
